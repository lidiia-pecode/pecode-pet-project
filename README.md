# Pecode Pet Project

This is a **Next.js** project using **TypeScript**.  
It also uses the following libraries:  
- **Material UI** (`@mui/material`) for UI components  
- **Zustand** for state management  
- **TanStack Query** (`@tanstack/react-query`) for data fetching  
- **Axios** for API requests  

The project will use [FakeStore API](https://fakestoreapi.com/) as a test data source.

## Getting Started

1. Install dependencies:
```bash
npm install
# or
yarn install
```
Run the development server:
```bash
npm run dev
# or
yarn dev
```
Open http://localhost:3000 in your browser.
